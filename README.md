# 🌍 Travel Sentiment Analytics

AI-powered sentiment analysis and trend dashboard for hotel/travel reviews.

## 📌 Features
- Sentiment classification (Positive / Negative / Neutral)
- Rating trend analysis by location
- NLP-based topic modeling
- Interactive visualizations

## 🛠️ Tech Stack
Python | Pandas | TextBlob | NLTK | Matplotlib | Seaborn | Scikit-learn

## 📂 Dataset
TripAdvisor Hotel Reviews — 515K reviews (Kaggle)
